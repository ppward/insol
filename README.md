## 📌 프로젝트 소개
**인솔(insol)**은 아이들의 안전한 귀가와 유치원과 학부모간의 커뮤니티 기능을 하는 플랫폼으로, 
맞벌이 가구의 증가와 돌봄 서비스의 수요가 지속적으로 증가함에 따라, 만 8세 미만의 자녀를 가진 학부모에게 유치원(어린이집) 간의 유기적인 소통을 위한 집중화 서비스를 목적으로 개발되었습니다.
