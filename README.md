# Start
1. git clone https://github.com/ppward/insol.git

    npm install 


2. 각 환경에 맞는 개발툴을 사용해 빌드 
    google map의 경우 api를 따로 발급 받아서 추가.
    안드로이드 스튜디오와 xcode의 환경에서 추가 
    ios - AppDelegate.mm 파일에 추가
